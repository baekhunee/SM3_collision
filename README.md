# 生日攻击

生日攻击指利用哈希函数发生碰撞的可能性，进行n次尝试直到找到一对碰撞的输入。

由于SM3输出为256比特，而个人笔记本不具备足够的算力，因而减少输出比特进行测试，最终成功攻击输出值为28比特的SM3函数。

# Rho攻击

将所有生成的hash值放入一条链中。随机生成字符串r，调用自定义的SM3函数计算哈希值，如果当前哈希值已经在链中，说明此时已经成环，找到了碰撞；如果当前哈希值不在链中，就将其添入。
